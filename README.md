# README

Rails project with a React and Redux front-end

- Ruby version 2.7

- System dependencies
  - webpacker 4.0
  - react-rails 2.6.1

#### Tip:

- run _./bin/webpack-dev-server_ in a separate window while coding to have any changes automatically build and reload the browser.
